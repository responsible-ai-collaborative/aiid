## Acerca de las taxonomías ##

Las taxonomías son aportadas a la base de datos de incidentes de IA por personas y organizaciones que trabajan para estructurar los datos y proporcionar vistas de los datos. Cada taxonomía debe tener la calidad y la exhaustividad suficientes para incluirse en la base de datos de incidentes de IA, pero las taxonomías son responsabilidad de las personas y organizaciones que las aportan.
